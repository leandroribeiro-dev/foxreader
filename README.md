\# FoxReader



Projeto inicial de uma ferramenta simples para auxiliar nos estudos por meio da leitura de textos.



\## Objetivo inicial



O FoxReader nasce como uma caixa de leitura em HTML, pensada para permitir que o usuário cole um texto e utilize o recurso nativo de leitura em voz alta do navegador, especialmente no Microsoft Edge.



A ideia principal é criar, aos poucos, uma ferramenta de apoio para:



\- leitura de textos de estudo;

\- revisão de materiais;

\- auxílio na concentração;

\- experimentação com recursos de leitura em voz.



\## Estado atual



Nesta primeira versão, o projeto possui apenas:



\- uma página HTML;

\- uma caixa editável para colar texto;

\- instrução para usar o recurso nativo de leitura do navegador;

\- estilização básica em CSS.



\## Tecnologias utilizadas



\- HTML5

\- CSS3



\## Como usar



1\. Abra o arquivo `FoxReader.html` no navegador.

2\. Cole um texto na caixa de leitura.

3\. Selecione o trecho desejado.

4\. Clique com o botão direito.

5\. Use a opção de leitura em voz alta do navegador.



\## Observação



Esta é a versão inicial do projeto. Novos recursos poderão ser adicionados gradualmente conforme a evolução dos estudos e testes.
